¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Define la estructura y el significado semántico del contenido.

Junto con CSS y JavaScript, forma la base de toda página web moderna.

Diseño Mobile First

El enfoque Mobile First consiste en escribir los estilos CSS pensando primero en pantallas pequeñas y luego ampliar con media queries para dispositivos más grandes.

Esto garantiza que la experiencia en móvil sea óptima desde el principio.

Tecnologías usadas

  • HTML5 semántico
  • CSS Mobile First
  • Flexbox (tarjetas)
  • CSS Grid (layout)
  • Media Queries ×2
  • Bootstrap 5 Navbar
